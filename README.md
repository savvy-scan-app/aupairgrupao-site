# AuPair Grupão - Website Oficial

Site oficial do AuPair Grupão para compliance com App Store da Apple.

## Páginas Incluídas

- **Home** (`index.html`) - Página inicial
- **Suporte** (`support.html`) - Central de suporte e FAQ
- **Política de Privacidade** (`privacy.html`) - Detalhes sobre coleta e uso de dados
- **Termos de Uso** (`terms.html`) - Termos e condições de uso
- **Diretrizes da Comunidade** (`community-guidelines.html`) - Regras da comunidade
- **Exclusão de Conta** (`account-deletion.html`) - Como excluir sua conta

## Tecnologias

- HTML5
- Tailwind CSS (via CDN)
- JavaScript vanilla
